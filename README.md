
Resolução de Exercícios - Programação I (Python)
[cite_start]Este repositório contém a resolução da lista de exercícios sobre definição e chamada de funções do curso de **Engenharia de Software**[cite: 6].

Conteúdo da Lista
[cite_start]O projeto aborda os seguintes requisitos solicitados no documento[cite: 10]:

[cite_start]**Manipulação Numérica**: Funções para contar dígitos e inverter números inteiros[cite: 11, 12].
[cite_start]**Tratamento de Datas**: Função para converter datas numéricas para o formato com mês por extenso[cite: 13].
[cite_start]**Interface em Terminal**: Função para desenhar molduras retangulares com dimensões ajustáveis[cite: 15, 16].
[cite_start]**Conversão de Unidades**: Sistema de menu para conversão entre escalas Celsius e Fahrenheit[cite: 17, 18].
[cite_start]**Recursividade**: Cálculo da série de Fibonacci utilizando chamadas recursivas[cite: 19, 21, 22].
[cite_start]**Matemática e Lógica**: Verificação de números perfeitos e listagem de intervalos[cite: 23, 25, 26].
[cite_start]**Sistemas de Horário**: Conversor de notação 24 horas para 12 horas (A.M./P.M.) com loop de repetição[cite: 27, 29, 32].
[cite_start]**Gestão Financeira**: Função `valor_pagamento` para cálculo de prestações com aplicação de multa e juros por atraso[cite: 33, 34, 39].

Sistema de Controle de Pedidos (Questão 10)
[cite_start]Um mini sistema **CRUD** (Create, Read, Update, Delete) desenvolvido para uma lanchonete[cite: 40, 48]. [cite_start]As funcionalidades incluem[cite: 50, 51]:

1.  [cite_start]**Cadastrar**: Registra cliente, item, quantidade e preço[cite: 52, 54].
2.  [cite_start]**Listar**: Exibe pedidos e calcula o valor total individual[cite: 56, 58].
3.  [cite_start]**Atualizar**: Permite modificar dados de pedidos existentes[cite: 59, 60].
4.  [cite_start]**Remover**: Exclui registros através de um identificador[cite: 61, 62].

Como Executar
1. Certifique-se de ter o Python 3 instalado.
2. Execute o arquivo principal:
   ```bash
   python exercicios_unifsa.py
   ```
3. [cite_start] O sistema da lanchonete continuará rodando até que a opção "Sair" seja selecionada no menu[cite: 68, 69].
